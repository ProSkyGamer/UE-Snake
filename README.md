# UE-Snake
Описание проекта:
Обычная змейка.
Что реализовано:
* Пошаговое движение змейки с поворотом
* Добавление хвоста при съедании "яблока"
* Поражение (возможность умереть об хвост/стены)
* Рандомный спавн яблока
* Счетчик съеденных яблок
